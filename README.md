# DNA Sequence Analyzer

A Python tool that analyzes DNA sequences.

## What it does
- Counts nucleotides (A, T, G, C)
- Calculates GC content
- Generates complement strand
- Generates reverse complement
- Transcribes DNA to mRNA
- Translates mRNA to Protein

## How to run
python dna-sequence-analyzer.py

## Sample Output
Nucleotide count: {'A': 3, 'T': 3, 'G': 3, 'C': 2}
GC content: 45.45 %
Complement: TACGTACGTAC
Reverse complement: CATGCATGCAT
mRNA: AUGCAUGCAUG
Protein: Met - His - Ala
